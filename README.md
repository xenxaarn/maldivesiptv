# Familyplusiptv

regularly updated IPTV playlist featuring international news, sports, kids & animation, and FIFA World Cup 2026 live streams.

Suitable for: VLC or any M3U-compatible player.


## Playlist Categories

The playlist is carefully organized into the following groups:

| Category | Description |
| :--- | :--- |
| News | International news (BBC, CNN, Al Jazeera, ABC, France 24, etc.) |
| Sports | Sports channels (ESPN, Arena Sport, FIFA+, beIN Sports, etc.) |
| Kids & Animation | Cartoons, anime, and kids' channels (Nickelodeon, Disney, etc.) |
| FIFA World Cup 2026 | Live 4K World Cup matches (Fox Sports 4K, FussBall TV) |
| Music | Music and radio channels |
| Business | Business and financial news |
| Global News Network | Regional news networks |
| Sports Networks | Specialized sports feeds |
## How to Use

### Option 1: Add the URL to your IPTV Player

Copy and paste this URL into any M3U-compatible player:

https://raw.githubusercontent.com/xenxaarn/familyplusiptv/main/familyplus.m3u

### For android it is highly recommended to use this opensourced app to play the m3u link Download the latest release in the repo :

https://github.com/oxyroid/M3UAndroid

- Open m3uandroid app "m3u"
- copy the link https://raw.githubusercontent.com/xenxaarn/familyplusiptv/main/familyplus.m3u
- click the "+" icon
- Paste clipbord or if you want to paste manually paste into the playlist link
- Click "SUBSCRIBE"
- you might need to restart your app and try it one more time

### For linux, macos, windows
- Open vlc app
- Media
- Open Network Streme
- Network
- paste https://raw.githubusercontent.com/xenxaarn/familyplusiptv/main/familyplus.m3u
- Press Ctrl + L to open the playlist

### For iphones under ios 18 vlc works for ios 26 or higher use NetIPTVPro
- Open vlc
- Browse
- Open Network Streme
- paste https://raw.githubusercontent.com/xenxaarn/familyplusiptv/main/familyplus.m3u
- Playlist 

## Updates

The playlist is updated regularly with new channels and links. Simply refresh your IPTV player to get the latest version.
